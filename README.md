# Python Basics Programs

This repository contains beginner-friendly Python programs created for practice.

## Programs Included

1. Simple Calculator
2. Even or Odd Number Checker
3. Prime Number Checker

## How to Run Programs

1. Open Command Prompt
2. Go to project folder:
   cd Desktop\python-basics

3. Run any program:
   py calculator.py
   py even_odd.py
   py prime_number.py

## Python Version
Python 3.10+
